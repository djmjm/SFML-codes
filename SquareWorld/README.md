Simple program/game developed with SFML c++ lib

# How to run
1) Install SFML lib.
- windows: https://www.sfml-dev.org/tutorials/2.5/start-vc.php
- linux: https://www.sfml-dev.org/tutorials/2.5/start-linux.php
2) Run makefile
3) Run squareworld.exec
